# wa-voicenote-triage

Personal WhatsApp voice-note triage via Twilio Sandbox + Azure OpenAI gpt-audio-1.5.

See [docs/PLAN.md](docs/PLAN.md) for the full design and implementation plan.

## Requirements
- Python 3.12
- uv
- ffmpeg
- Docker (for local CI mirror)

## Quickstart
```bash
make install
make ci
```
